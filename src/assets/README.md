# Assets Directory

This directory contains static assets that are bundled with the application.

## Structure

- `fonts/`: Custom fonts and typography files.
- `icons/`: SVG icons and small graphical elements.
- `images/`: General images, photos, and backgrounds.
  - `agents/`: Valorant agent portraits and icons.
  - `maps/`: Valorant map images and minimaps.
  - `ranks/`: Competitive rank icons.
  - `weapons/`: Weapon skins and icons.
- `videos/`: Local video files for backgrounds or animations.

## Usage

You can import these assets using the `@` alias:

```javascript
import HeroImage from "@/assets/images/hero-bg.png";
```
